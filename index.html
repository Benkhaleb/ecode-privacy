Politique de confidentialité – ECode

Dernière mise à jour : 16 décembre 2025

1. Collecte des données

L’application ECode collecte certaines informations nécessaires à son fonctionnement, notamment les données de connexion, les informations liées à l’utilisation de l’application et les données pédagogiques.

2. Utilisation des données

Les données collectées sont utilisées uniquement pour :

  • le fonctionnement de l’application
  • l’amélioration des services
  • la gestion des utilisateurs

3. Partage des données

Aucune donnée personnelle n’est vendue ou partagée avec des tiers.

4. Sécurité

Les données sont stockées de manière sécurisée via des services cloud conformes aux standards de sécurité.

5. Contact

Pour toute question, vous pouvez nous contacter à : contact@ecode.app
